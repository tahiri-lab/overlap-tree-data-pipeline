# Reference-tree-based datasets of phylogenetic trees defined on different but overlapping sets of taxa (Mode 2)

This folder contains the datasets constructed by the proposed data pipeline in Mode 2 (reference-tree-based).

Constructed datasets include four species groups: **Amphibians, Mammals, Sharks, and Squamates**. The source phylogenetic data were obtained from **VertLife**.

In addition to the prepared datasets, this folder also contains the script used to validate the datasets.

## Source

- **VertLife data portal**: [https://vertlife.org/data/](https://vertlife.org/data/)
- **VertLife direct data download page (for reference trees)**: [https://data.vertlife.org/](https://data.vertlife.org/)
